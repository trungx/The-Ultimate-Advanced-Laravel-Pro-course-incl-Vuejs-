In the previous lesson, we used the map function to query our database and get all completed lessons for a series. I'll like to show you another approach to doing this, which might drastically reduce the number of queries we make to the database.

Here is the new solution:

/**
 * Get all completed lessons for a series
 *
 * @param [Bahdcasts\Series] $series
 * @return \Illuminate\Support\Collection(Bahdcasts\Lesson)
 */
 public function getCompletedLessons($series) {
 // 1, 2, 4
     return Lesson::whereIn('id', 
         $this->getCompletedLessonsForASeries($series)
     )->get();
 }


Using a `whereIn` would make just one query to the database, thus fetching all the completed lessons, even if there are 100 of them. 

This would greatly optimize our application performance.
